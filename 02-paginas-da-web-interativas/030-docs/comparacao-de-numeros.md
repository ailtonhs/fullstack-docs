# Comparando Números

## Comparação de Números

Podemos usar comparações para verificar se um número é menor ou maior que outro número.

Para verificar se um número é menor que outro número, usamos o **operador de menor que**, **<**.

```js
console.log(1 < 90);
```
Se o número à esuerda for menor que o número à direita, como em **1 < 235**, o resultado é **true**.

```js
console.log(1 < 235);
```
Se o número à esquerda não for menor que o número à direita, como em **235 < 1**, o resultado é **false**

```js
console.log(235 < 1);
```
Para verificar se um número é maior que outro número número, usamos o **operador maior que**, **>**.

```js
console.log(101 > 90);
```
